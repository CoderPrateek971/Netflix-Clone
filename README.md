# Netflix Landing Page Clone

A fully responsive Netflix landing page clone built using only **HTML** and **CSS**.

## Features
- Pure HTML & CSS (no JavaScript)
- Mobile responsive design
- Clean and modern UI
- Background overlay with video alignment
